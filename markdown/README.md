# Markdown 문법 정리
## 헤더
* 총 6개의 헤더가 존재한다.
* #이 늘어날 때마다 제목의 수순은 내려간다.
해더|실행결과
---|---
#h1|# h1
##h2|## h2

---
## 리스트
* 도형과 수나 알파벳으로 표기된다.
  1. 도형 
  ```
  * 내용1
  * 내용2
     * 하위내용1
     * 하위내용2
  ```
  * 내용1
  * 내용2
     * 하위내용1
     * 하위내용2
     
  2. 수와 도형
  ```
  1. 내용1
     * 하위내용1
  ```
  1. 내용1
     * 하위내용1
  
  3. 수와 알파벳
  ```
  1. 내용1
     1. 하위내용1
     2. 하위내용2
   ```
  1. 내용1
     1. 하위내용1
     2. 하위내용2
